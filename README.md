# 대만 4일 여행 일정표

## 🌟 프로젝트 소개

Apple 웹사이트 스타일의 미니멀하고 고급스러운 대만 여행 일정표 웹사이트입니다. 모바일 우선 반응형 디자인으로 구현되어 있으며, 부드러운 애니메이션과 실용적인 기능들을 포함하고 있습니다.

## 🎨 디자인 특징

- **Apple 스타일 디자인**: 미니멀한 UI, 고급스러운 여백, 중성색 기반 컬러 팔레트
- **모바일 우선 반응형**: 모든 기기에서 최적화된 사용자 경험
- **SF Pro, Noto Sans 기반 타이포그래피**: 깔끔하고 가독성 높은 폰트
- **부드러운 애니메이션**: fade/slide 효과로 자연스러운 전환
- **심플한 아이콘**: Font Awesome 아이콘 활용

## 🔧 주요 기능

### ✅ 1. Day 1-4 여행 일정 표시
- 카드 형태의 일정 표시
- 도착시간 / 체류시간 / 출발시간 표시
- 메모 및 카테고리별 색상 구분
- 이동 방식과 이동 시간 표시 (도보, MRT, 버스 등)

### ✅ 2. Google Maps 연결
- 각 장소 클릭 시 Google Maps로 연결
- 대만 현지어 장소명으로 정확한 위치 검색
- 새 창으로 열기

### ✅ 3. 환율 계산기 (TWD ↔ KRW)
- 실시간 환율 계산 (1 TWD = 42.5 KRW 기준)
- 양방향 계산 지원
- 플로팅 버튼으로 쉬운 접근

### ✅ 4. 날씨 표시
- 네이버 날씨 타이베이 페이지 iframe으로 통합
- 플로팅 버튼으로 쉬운 접근

### ✅ 5. Day 간 전환
- 상단 탭으로 Day 간 쉬운 전환
- 모바일에서는 스와이프로도 전환 가능
- 부드러운 애니메이션 효과

## 📱 반응형 지원

- **모바일 (max-width: 480px)**: 최적화된 모바일 레이아웃
- **태블릿 (max-width: 768px)**: 적응형 레이아웃
- **데스크톱**: 전체 기능 지원

## 🚀 사용 기술

- **HTML5**: 시맨틱 마크업
- **CSS3**: 모던 CSS, Flexbox, Grid
- **JavaScript (ES6+)**: 클래스 기반 애플리케이션
- **Font Awesome**: 아이콘 라이브러리
- **Google Fonts**: Noto Sans KR 폰트

## 🎯 카테고리별 색상

- **음식점/디저트**: 🟢 녹색 계열
- **교통/출국/입국/비행**: 🔵 파란색 계열
